# BoxTable JSON 配置类型定义

## 整体配置结构

```typescript
{
  // 基础配置
  tableKey?: string                    // 表格唯一标识（用于保存列设置）
  queryApi: Function                  // 数据查询接口（必填）
  apiList?: string                    // 用于取数据的数组
  exportApi?: Function                // 导出接口（可选）
  exportFileName?: string             // 导出文件名
  size?: 'large' | 'default' | 'small' // 表格密度
  pageSize?: number                   // 每页显示条数
  pagination?: boolean                // 是否显示分页
  page?: number                       // 当前页码
  total?: number                      // 数据总数
  
  // 查询表单配置
  columnsPerRow?: number              // 每行显示几个查询项（默认3）
  defaultExpand?: boolean             // 是否默认展开查询表单
  
  // 查询项列表
  queryList: Array<QueryItem>         // 查询表单配置
  
  // 表格列配置
  tableColumn: Array<TableColumn>     // 表格列配置
}
```

## 查询项类型 (QueryItem)

### 基础类型
```typescript
{
  type?: 'text' | 'select' | 'cascader' | 'daterange' | 'remote',  // 表单类型
  field: string,                     // 字段名（对应接口参数）
  label: string,                     // 显示标签
  defaultValue?: any,                // 默认值
  isSlot?: boolean,                  // 是否使用插槽
  slotName?: string                  // 插槽名称（isSlot为true时使用）
}
```

### 文本输入框 (text)
```typescript
{
  type: 'text',
  field: string,
  label: string,
  defaultValue?: string
}
```

### 下拉选择器 (select)

#### 静态数据
```typescript
{
  type: 'select',
  field: string,
  label: string,
  option: Array<{ label: string, value: any }>,  // 选项列表
  defaultValue?: any
}
```

#### 远程数据
```typescript
{
  type: 'select',
  field: string,
  label: string,
  queryApi: Function,                // 数据接口函数
  props?: {                          // 字段映射
    value: string,                   // 值字段名
    label: string                    // 标签字段名
  },
  lazy?: boolean,                    // 是否懒加载（聚焦时才加载）
  queryParams?: any,                 // 固定查询参数
  formatData?: Function,             // 数据格式化函数
  defaultValue?: any
}
```

#### 字典数据
```typescript
{
  type: 'select',
  field: string,
  label: string,
  dictType?: string,                 // 字典类型
  dictApi?: Function,                // 字典接口（可选）
  lazy?: boolean,
  defaultValue?: any
}
```

### 级联选择器 (cascader)
```typescript
{
  type: 'cascader',
  field: string,
  label: string,
  lazy?: boolean,                    // 是否懒加载
  queryApi?: Function,               // 树形数据接口
  props?: {                          // 级联配置
    value: string,                   // 值字段名
    label: string,                   // 标签字段名
    children: string,                // 子节点字段名
    checkStrictly?: boolean,         // 是否严格选择
    multiple?: boolean,              // 是否多选
    emitPath?: boolean               // 是否返回完整路径
  },
  defaultValue?: any[]               // 默认值（数组）
}
```

### 日期范围选择器 (daterange)
```typescript
{
  type: 'daterange',
  field: string,
  label: string,
  defaultValue?: [string, string]    // [开始日期, 结束日期]
}
```

### 远程搜索框 (remote)
```typescript
{
  type: 'remote',
  field: string,
  label: string,
  queryApi: Function,                // 搜索接口
  props?: {                          // 字段映射
    value: string,
    label: string
  },
  debounceTime?: number,             // 防抖延迟（毫秒，默认300）
  minSearchLength?: number,          // 最小搜索长度（默认2）
  maxResults?: number,               // 最大返回结果数（默认20）
  skipEmptySearch?: boolean,         // 是否跳过空搜索（默认true）
  showTipOnShortLength?: boolean,    // 搜索长度不足时是否显示提示
  defaultFirstOption?: boolean,      // 是否默认选中第一个选项
  reserveKeyword?: boolean,          // 选中后是否保留搜索关键字
  queryParams?: Function,            // 动态参数函数 (keyword) => params
  formatData?: Function,             // 自定义格式化函数
  defaultValue?: any
}
```

### 自定义插槽 (isSlot)
```typescript
{
  field: string,
  label: string,
  isSlot: true,
  slotName?: string,                 // 不填则使用 field
  defaultValue?: any
}
```

## 表格列类型 (TableColumn)

### 多选框列
```typescript
{
  type: 'selection',
  width?: string | number,           // 列宽（默认'55'）
  fixed?: 'left' | 'right'           // 固定位置
}
```

### 序号列
```typescript
{
  type: 'index',
  width?: string | number,           // 列宽（默认'80'）
  label?: string,                    // 列标题（默认'序号'）
  fixed?: 'left' | 'right'
}
```

### 普通列
```typescript
{
  prop: string,                      // 数据字段（支持 a.b.c 格式）
  width?: string | number,           // 列宽
  label: string,                     // 列标题
  minWidth?: string | number,        // 最小宽度
  fixed?: 'left' | 'right',          // 固定位置
  sortable?: boolean | 'custom',     // 是否可排序
  showOverflowTooltip?: boolean,     // 超出是否显示提示（默认true）
  align?: 'left' | 'center' | 'right' // 对齐方式
}
```

### 自定义插槽列
```typescript
{
  prop: string,                      // 数据字段
  width?: string | number,
  label: string,
  isSlot: true,                      // 启用插槽
  slotName?: string,                 // 插槽名称（不填则使用 prop）
  fixed?: 'left' | 'right',
  sortable?: boolean,
  showOverflowTooltip?: boolean
}
```

### 操作列（通常使用）
```typescript
{
  fixed: 'right',                    // 通常固定在右侧
  label: '操作',
  width?: string | number,           // 建议150-200
  isSlot: true,
  slotName: 'operation'              // 操作列插槽名称
}
```

## 字段说明

### 查询表单字段说明
- `field`: 对应接口查询参数的字段名
- `defaultValue`: 根据类型不同，默认值格式也不同
  - `text`: `string`
  - `select`: `any` (通常为字符串或数字)
  - `cascader`: `any[]` (数组，表示级联路径)
  - `daterange`: `[string, string]` (开始日期, 结束日期)
  - `remote`: `any`

### 表格列字段说明
- `prop`: 支持点号分隔的路径，如 `deptId.name` 会解析为 `row.deptId.name`
- `width`: 可以是数字或字符串，如 `120` 或 `'120px'`
- `isSlot`: 为 `true` 时，该列使用插槽渲染
- `slotName`: 插槽名称，在父组件中使用 `#slotName="{ row }"`

### 接口函数要求
- `queryApi`: 接收参数 `{ page, pageSize, ...searchParams }`，返回 `{ code: 200, data: { list: [], total: number } }`
- `exportApi`: 接收搜索参数，返回文件流
- 远程接口：返回数组格式数据

## 使用建议

1. **命名规范**
   - `tableKey`: 使用 `模块名_table` 格式，如 `user_table`
   - `field`: 使用驼峰命名，与接口参数一致
   - `slotName`: 使用有意义的名称，如 `status`, `operation`

2. **性能优化**
   - 远程数据接口使用 `lazy: true` 避免初始加载
   - 复杂查询使用防抖和最小搜索长度
   - 大量数据时使用虚拟滚动（可扩展功能）

3. **可维护性**
   - 将配置单独放在 `config.ts` 文件中
   - 使用 TypeScript 类型检查
   - 添加注释说明特殊配置

4. **扩展性**
   - 通过插槽实现自定义渲染
   - 可扩展新的表单类型
   - 支持自定义工具栏按钮
