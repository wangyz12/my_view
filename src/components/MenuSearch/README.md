# 菜单搜索组件 (MenuSearch)

一个美观、功能强大的菜单搜索组件，支持快速搜索和跳转到菜单路由。

## 🎯 功能特性

- 🔍 **智能搜索**: 支持菜单标题和路径搜索
- 🎨 **美观UI**: 现代化设计，支持主题切换
- ⌨️ **键盘导航**: 支持键盘快捷键操作
- 📱 **响应式**: 适配不同屏幕尺寸
- ⚡ **实时过滤**: 输入时实时显示搜索结果
- 🚀 **快速跳转**: 点击搜索结果快速跳转

## 📁 组件位置

组件已集成到头部导航右侧区域，位于全屏按钮之前：

```
头部导航右侧布局：
[🔍 菜单搜索] [🖥️ 全屏] [⚙️ 设置] [👤 用户信息]
```

## 🚀 快速开始

组件已自动注册，无需额外配置即可使用：

```vue
<template>
  <menu-search />
</template>
```

## 🎨 设计特点

### 1. 交互设计
- **折叠/展开**: 默认显示搜索按钮，点击后展开输入框
- **动画效果**: 平滑的展开/折叠动画
- **视觉反馈**: 悬停效果、选中状态

### 2. 搜索功能
- **实时搜索**: 输入时实时显示结果
- **智能过滤**: 支持标题和路径搜索
- **结果限制**: 最多显示10个结果
- **外部链接**: 支持外部链接识别

### 3. 键盘支持
- **↑/↓**: 在结果列表中导航
- **Enter**: 选择当前选中项
- **Esc**: 关闭搜索框
- **F2**: 快速聚焦搜索框（可选）

## 🔍 搜索算法

### 匹配规则
1. **标题匹配**: 优先匹配菜单标题
2. **路径匹配**: 其次匹配路由路径
3. **拼音支持**: 可扩展支持拼音搜索
4. **模糊搜索**: 支持部分匹配

### 排序规则
1. **标题完全匹配**（最高优先级）
2. **标题开头匹配**
3. **标题包含匹配**
4. **路径匹配**

## ⚠️ 注意事项

### 1. 数据依赖
组件依赖 `userStore.menus` 数据，确保菜单数据已加载

### 2. 权限控制
搜索结果会自动过滤隐藏菜单和按钮类型菜单

### 3. 外部链接
外部链接会在新窗口或当前窗口打开

### 4. 键盘冲突
注意避免与页面其他快捷键冲突
