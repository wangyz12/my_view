# CesiumEarth 组件快速使用指南

## 功能概述
- 🚁 3架无人机自动巡逻
- 🚨 随机报警触发
- 📍 智能位置记忆与恢复
- 🎨 呼吸效果报警区域

## 快速开始

### 1. 启动项目
```bash
npm run dev
```

### 2. 访问页面
打开浏览器访问：`http://localhost:5174/#/dataview`

### 3. 基本操作

#### 开始巡逻
1. 点击"起飞"按钮
2. 3架无人机开始按照预设路线巡逻
3. 每80秒完成一圈巡逻

#### 选择无人机
1. 点击地图上的任意无人机
2. 右侧信息面板显示该无人机实时信息：
   - 名称和电量
   - 经纬度坐标
   - 飞行高度
   - 当前状态

#### 触发报警测试
1. 点击"随机报警"按钮（红色）
2. 观察以下变化：
   - 地图随机位置出现**淡红色呼吸区域**
   - 左上角显示报警信息面板
   - 最近的无人机自动飞往报警区域

#### 报警处理流程
1. **无人机响应**：最近的无人机飞往报警区域（5秒）
2. **报警解除**：到达后红色呼吸区域**立即消失**
3. **智能返回**：无人机返回**原来的巡逻位置**（3秒）
4. **继续巡逻**：从中断位置继续巡逻任务

## 界面说明

### 控制面板（右下角）
- 🛫 **起飞**：开始/重新开始巡逻
- ⏸️ **暂停**：暂停所有无人机
- 🔄 **重置**：重置所有无人机到起点
- 🚨 **随机报警**：触发随机报警测试

### 信息面板（右下角）
显示选中无人机的实时信息：
- 名称和电量
- 经纬度坐标（6位小数精度）
- 飞行高度（米）
- 当前状态（正常巡逻/前往报警/返回等）

### 报警面板（左上角）
报警时显示：
- 🚨 报警位置（经纬度）
- 处理无人机名称
- 报警状态（等待响应/正在前往/已解除等）

## 核心特性

### 1. 智能位置记忆
- **记录时机**：无人机离开巡逻路线响应报警时
- **记忆内容**：精确的3D位置和巡逻进度
- **恢复精度**：毫米级精度返回原位置

### 2. 视觉反馈
- **呼吸报警区域**：淡红色，透明度动态变化
- **实时状态更新**：所有面板信息实时刷新
- **平滑动画**：无人机移动平滑自然

### 3. 自动化处理
- **自动选择**：系统自动选择最近的无人机
- **自动恢复**：报警处理后自动恢复巡逻
- **自动循环**：巡逻完成自动重新开始

## 测试场景

### 场景1：基本巡逻测试
1. 点击"起飞"
2. 观察3架无人机按不同路线巡逻
3. 点击不同无人机查看实时信息
4. 点击"暂停"观察停止效果
5. 点击"重置"回到初始状态

### 场景2：单次报警测试
1. 点击"起飞"开始巡逻
2. 等待无人机巡逻10-20秒
3. 点击"随机报警"
4. 观察完整处理流程：
   - 报警区域出现
   - 无人机飞往报警区域
   - 到达后报警区域消失
   - 无人机返回原位置
   - 继续巡逻

### 场景3：连续报警测试
1. 开始巡逻后，连续点击"随机报警"2-3次
2. 观察系统处理多个报警的能力
3. 注意：一次只能处理一个报警

## 故障排除

### 问题1：无人机不移动
- 检查是否点击了"起飞"
- 检查浏览器控制台是否有错误
- 尝试点击"重置"后重新开始

### 问题2：报警区域不消失
- 确保无人机已到达报警区域
- 检查报警面板状态是否更新
- 刷新页面重新测试

### 问题3：性能问题
- 减少浏览器标签页数量
- 关闭不必要的浏览器扩展
- 确保网络连接稳定

## 技术栈
- **前端框架**：Vue 3 + TypeScript
- **3D引擎**：Cesium.js
- **UI组件**：Element Plus
- **构建工具**：Vite

2. **位置精度**：返回位置可能有微小误差
3. **浏览器兼容**：建议使用Chrome或Edge最新版
4. **性能要求**：需要支持WebGL的显卡
